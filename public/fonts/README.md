# Orange Avenue (premium display font)

Add your licensed webfont files here:

- `OrangeAvenue-Regular.woff2` (required)
- `OrangeAvenue-Regular.woff` (optional fallback)

The site uses **Cormorant Garamond** from Google Fonts until these files are present.

Convert `.otf` / `.ttf` to `.woff2` with [Transfonter](https://transfonter.org/).
